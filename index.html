VtM character sheet app
Create and share Vampire the Maquerade character sheets. The app lets you adjust the sheet template by adding, removing or renaming variable traits. Furthermore, the app accounts for trait limit changes due to lower character generations. This lets you customize your character sheets in ways no printable version can.

Multiplayer Carcassonne
Play a game of Carcassonne with a bunch of friends. The app uses the game’s basic set of tiles and allows up to six players per game. The app doesn’t automatically track the score leaving it to the players but it does validate tile placements. In rare cases where a tile cannot be played, the app lets the player draw a new tile.

Scry bot
This bot leverages Telegram and Scryfall APIs to allow Telegram users to fetch card data from the Scryfall database. The information that the bot retrieves includes a card image, either latest or specific to the user’s request, as well as format legality info and card price in euro. This can be handy in Magic-themed group chats and elsewhere.

Draft simulators
A series of web applications designed to let users practice drafting in Magic the Gathering and Eternal Card Game against an algorithm. In each case the algorithm in question would be the principal difference between the implementations. The first two apps were designed for Magic the Gathering. The latter was built for Eternal Card Game and featured a UI reminiscent of the one implemented in the actual game. This is the version currently available on Heroku.
